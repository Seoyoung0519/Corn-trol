# 🌽 Corn-trol


**AI 기반 사고 흐름 연결 및 집중 사고 지원 서비스**
<br><br>
> 짧게 흩어진 생각들을 기록하고,<br>AI 기반 연결을 통해 하나의 사고 흐름으로 확장하는 서비스

---

## 프로젝트 소개

Corn-trol은 빠르게 흩어지는 생각과 아이디어를 기록하고,  
AI 기반 연결을 통해 하나의 사고 흐름으로 확장할 수 있도록 돕는 사고 흐름 관리 서비스입니다.

기존 메모 앱이 단순 기록 저장에 집중했다면,  
Corn-trol은 사용자가 남긴 텍스트·음성 기록 간의 의미적 관계를 분석하여  
사고의 연결성과 확장성을 지원하는 것을 목표로 합니다. 

서비스는 크게 세 가지 흐름으로 구성됩니다.

- 떠오른 생각을 즉시 기록하는 **팝콘 수집기**
- 기록된 사고 조각들을 AI가 연결하는 **알곡 꿰기**
- 하나의 생각에 깊게 몰입하도록 돕는 **알곡 식히기**

이를 통해 사용자가 흩어진 아이디어를 단순 저장하는 것을 넘어,  
하나의 논리적인 사고 흐름으로 발전시킬 수 있도록 지원합니다. 

---

## 문제 정의

현대의 디지털 환경은 숏폼 콘텐츠와 빠른 정보 소비 중심으로 변화하면서  
사용자의 집중력과 사고 지속성을 약화시키고 있습니다. 

특히 브레인스토밍이나 창의적 사고 과정에서는  
다양한 생각이 순간적으로 떠오르지만 이를 체계적으로 연결하거나  
하나의 결과물로 발전시키기 어려운 문제가 존재합니다.

기존 메모 앱은 기록 저장 기능에는 강점을 가지지만,  
기록 간 관계를 사용자가 직접 정리해야 하기 때문에  
사고 흐름이 쉽게 단절된다는 한계가 있습니다.

Corn-trol은 이러한 문제를 해결하기 위해  
AI 기반 의미 분석을 활용하여 기록들을 연결하고,  
사고를 하나의 흐름으로 구조화하는 경험을 제공하고자 합니다.

또한 단순 앱 차단 중심의 집중 서비스와 달리,  
사용자가 특정 사고에 깊게 몰입하고 사고를 확장할 수 있도록 지원하는 데 초점을 두고 있습니다.

## 주요 기능

### 1️⃣ 팝콘 수집기
> 떠오르는 생각을 즉시 기록하는 사고 수집 기능

#### 주요 기능
- 텍스트 및 음성 기반 기록 입력 지원
- 기록 데이터 자동 저장
- 핵심 키워드 자동 태깅
- 아이디어 휘발 방지를 위한 빠른 기록 흐름 제공

---

### 2️⃣ 알곡 꿰기
> 흩어진 사고 조각들을 연결해 하나의 사고 흐름으로 구조화하는 기능

#### 주요 기능
- AI 기반 유사 사고 연결 추천
- 키워드 및 문맥 기반 기록 연결
- 사고 흐름 시각화
- 산발적인 아이디어 구조화
- 연결된 기록 기반 사고 확장 지원

---

### 3️⃣ 알곡 식히기
> 하나의 생각에 몰입하도록 돕는 사고 집중 모드

#### 주요 기능
- 타이머 기반 사고 몰입 기능
- 외부 자극 차단 환경 제공
- AI 질문 기반 사고 확장 유도
- 주의력 회복 및 깊은 사고 지원
- 특정 사고 흐름에 대한 지속적 집중 환경 제공

---

### 4️⃣ AI 기반 사고 연결 시스템
> 기록 간 의미적 관계를 분석하여 사고 흐름을 연결하는 핵심 기능

#### 주요 기능
- AI 기반 기록 유사도 분석
- 의미 기반 사고 연결 추천
- 키워드 및 문맥 기반 연결 구조 생성
- 사고 흐름 확장 및 창의적 연결 지원

---
